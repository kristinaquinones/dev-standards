## Example

**Campaign:** Product Launch Campaign  
**Goal:** Generate 200 signups in first month

**Target Audience:** Solo founders building their first product

**Positioning Statement:**
For solo founders building their first product, TaskTracker is a project management tool that helps them stay organized without complexity. Unlike Trello or Asana, TaskTracker focuses on simplicity and speed.

**User Story 1:**
```
As a solo founder,
I want to quickly see what tasks I need to work on today,
So that I can focus on building instead of managing my task list.
```

**How this relates to campaign message:**
This story emphasizes speed and simplicity - core messages for the campaign. The campaign should highlight how TaskTracker lets founders see their tasks instantly without complex setup.

**Core Message:**
TaskTracker helps solo founders stay organized and focused without the complexity of traditional project management tools.

**Supporting Message 1: Simple and Fast**
Get started in minutes, not hours. No complex setup or training required.
- **Connected to User Story:** User Story 1 (quickly see tasks)
- **Benefit:** Saves time, reduces friction

**Content Plan:**
- Blog post: "How Solo Founders Can Stay Organized Without Complexity" - supports core message and Supporting Message 1
- Demo video: Quick 2-minute setup walkthrough - supports Supporting Message 1
- Twitter thread: Key features with focus on speed - supports Supporting Message 1
