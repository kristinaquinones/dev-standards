## Example

### Level 1: Core Message
TaskTracker helps solo founders stay organized and focused without the complexity of traditional project management tools.

### Level 2: Supporting Messages

**Supporting Message 1: Simple and Fast**
Get started in minutes, not hours. No complex setup or training required.

**Supporting Message 2: Built for Solo Founders**
Designed specifically for people building products alone, not teams.

**Supporting Message 3: Focus on What Matters**
Only the features you need, without the bloat of enterprise tools.

**Supporting Message 4: Affordable**
Priced for bootstrapped founders, not enterprise budgets.

**Supporting Message 5: Grows With You**
Start simple, add complexity only when you need it.

### Level 3: Proof Points

**For Supporting Message 1 (Simple and Fast):**
- Average setup time: 5 minutes
- No onboarding required
- Intuitive interface that requires no training

**For Supporting Message 2 (Built for Solo Founders):**
- Designed based on interviews with 50+ solo founders
- Features prioritized for individual use cases
- Workflows optimized for one-person teams

[Continue for each supporting message...]

## Usage Tips

- Keep core message simple and memorable
- Supporting messages should work together to tell a complete story
- Proof points make messages credible; use data, examples, or stories
- Adapt proof points for different audiences or use cases
- Test messages with target users to see what resonates

This template is typically used as part of a [Campaign Brief](campaign-brief-template.md), which includes positioning, user stories, messaging hierarchy, and content planning.
