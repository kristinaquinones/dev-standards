# Product Templates

This directory contains reusable templates for product management documentation.

## Available templates

1. [Product & Design Brief Template](product-design-brief-template.md) - Unified template for product requirements and design briefs
2. [User Story Template](user-story-template.md) - User story format and examples

## Using templates

- Copy templates and customize for your specific needs
- Templates are starting points; adapt as necessary
- Include completed examples where helpful
- Link to templates from relevant [product standards](../README.md)

## Contributing templates

To add new templates:

1. Create template file in this directory
2. Add entry to this README index
3. Link to template from relevant product standards files
4. Include examples and usage guidance
